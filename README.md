# Chorong Lang

One of useless languages. **_Chorongchorong!_**

## Code Table

The storage is `stack`.

|  Character   | Action                                                |
| :----------: | :---------------------------------------------------- |
|      규      | Insert `0` on the top of the stack                    |
|      귯      | `TOP` += 1                                            |
|      류      | `TOP` << 1                                            |
|      륫      | `TOP` -= 1                                            |
| `[New Line]` | `POP` the stack and print the popped data until empty |

## Interpreter

Works with `node`.

```bash
$ ./chorong.js helloworld.chr
```
